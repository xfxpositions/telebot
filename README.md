# Xfxpositions Telebot

## Build

`build.bat`

## Installation:

### Download the binary

```
- Go to the releases
- Download the setup.exe
- Execute it
- Done.
```

### Run from source

```sh
pip3 install -r requirements.txt
```

## Running

### Cli

```sh
python main.py
```

### Gui

```
python gui.py
```

- only works in windows 10+
