# Logo Telebot DEMO

## Warning: This repository is private, if you see this in somewhere public. Please email to yusuf.karaca@logo.com.tr

## Installation:

### Download the binary

```
- Go to the releases
- Download the setup.exe
- Execute it
- Done.
```

### Run from source

```sh
pip3 install -r requirements.txt
```

## Running

### Cli

```sh
python main.py
```

### Gui

```
python gui.py
```

- only works in windows 10+
